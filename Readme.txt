== Description ==
Display post by drop-down or list by category or categories using Widgets.

= Features =

* Add the custom text as a widget title
* Select the specific category to show the post on widget
* Select the number of post to show with ranges from 1 to 10 or all posts
* Restrict title length of the post
* Select Multiple Categories
* Show post in Drop down option (combo box) or List view.
* Show/Hide Bullet style on list view
* Customize the default text "Please Select" in Drop down option.
* Sort the Post by Ascending, Descending, Random, post order and Post Modified

Free Version  : [Category Widget Support](https://buffercode.com/project/category-widget/)

Pro Version  : [Category Widget Pro Support](https://ifecho.com/category-widget-pro)


== Installation ==

1. Upload the "category-widget" directory to the plugins directory.
2. Go to the plugins setting page and activate "Category Widget"
3. Go to Appearance --> Widget --> Drag BC Category Widget to appropriate location
4. Choose the desired options.
